Após liberar uma atualização que supostamente causava problemas com o computador, a Microsoft veio a público dizer que é mentira que o Windows está passando por problemas. Segundo a empresa, o Windows 7 é inteligente e possui um sistema de segurança que permite a autodestruição antes que qualquer vírus o infecte.Os usuários que tiveram o sistema autodestruído deverão não só agradecer, mas também pagar a quantia de 99 dólares para a empresa devido à utilização do sistema de proteção. Os usuários que se recusarem efetuar o pagamento serão rastreados e cobrados tão logo sejam encontrados. A previsão de Windows 7 voltar a rodar é nula, segundo Bill Gates, executar o Windows é deixá-lo disponível a vírus, é melhor que fique assim como está, é muito mais seguro.Vinícius Antunes
