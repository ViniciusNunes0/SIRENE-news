O confronto entre Brasil e Colômbia está sendo tratado pela imprensa esportiva mundial como o duelo “Joelma x Shakira” transferindo para os gramados a rivalidade que existe entre as duas divas mundiais.Muitos achavam que o duelo seria marcado pela rivalidade entre os craques James x Neymar, mas a imprensa esportiva preferiu destacar outras rivalidades entre os dois países. Veja outros apelidos que a imprensa mundial deu para o clássico sul-americano:Beira-Mar x Escobar;Jorge Amado x Garcia Marquez;Malhação x Betty, a feia;Chiquititas x Isa TKM;David Luis x Valderrama;Massa x Montoya;Café mais-ou-menos x Café bom.
